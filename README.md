# Dota Analytics Platform

Plataforma analítica **end-to-end** para **Dota 2**, construída com foco em **engenharia de dados, MLOps e ML aplicado**, utilizando dados reais da API pública do **OpenDota**.

O projeto simula uma **plataforma de dados em produção**, evoluindo desde ingestão e banco analítico até **modelos de machine learning expostos via API**, com módulos de recomendação e predição.

---

A plataforma foi desenhada para permitir a evolução natural de:
- Analytics → ML → Produto
- Batch → API
- Dados brutos → features → decisões
